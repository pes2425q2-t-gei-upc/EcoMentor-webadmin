# Admin Dashboard

## Live Demo
[Deployed webpage](https://ecomentor-admin.onrender.com/)

## Features
- **User Management:** Create, edit, and manage user accounts and permissions
- **Content Management:** Organize and update system content
- **Data Visualization:** View key metrics and analytics
- **System Settings:** Configure application parameters

## Tech Stack
- React Admin
- Material UI
- RESTful API Integration

## Setup
1. Clone repository
2. Run `npm install`
3. Set environment variable `VITE_API_URL`
4. Run `npm run dev`

## Access
Login with admin credentials to access the dashboard features.
